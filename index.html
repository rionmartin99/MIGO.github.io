MIGO User Guide

Connecting

Turn on MIGO, then tap the Connect button on the left. Select "Migo_Web_Bot" from the popup menu.

Coding Basics

Drag blocks from the left menu into the workspace. Your code must be snapped inside the Setup block (runs once) or the Loop block (runs forever) to work.

Hardware Plugs

MIGO uses simple Plugs instead of complicated pin numbers. Connect your wires to these pins on the ESP32 board:

  • Plug A (Pin 34): Use for sensors (Buttons, Knobs, Light Sensors).
  • Plug B (Pin 35): Use for sensors.
  • Plug C (Pin 32): Use for sensors.
  • Plug D (Pin 14): Use for outputs (LEDs, Buzzers).
  • Ultrasonic: Automatically uses Trigger (Pin 13) and Echo (Pin 12).

Running Your Code

Click Upload to send the code to MIGO. It will save it to memory so you never lose it! Then click Run to watch MIGO execute your commands.

Block Library Reference

Starting Events

  • Setup: Code placed here runs exactly once when MIGO starts. Great for setting up pins or initial variables.
  • Loop forever: Code here repeats continuously until you turn MIGO off. This is where your main brain logic goes!

Logic & Control

  • If / Do: Makes a decision. If a condition is true, it does the action inside. Click the blue gear icon on the block to add 'Else' or 'Else If' branches!
  • Compare (=, <, >): Used inside 'If' blocks to compare two values (e.g. Distance < 10).
  • Repeat X times: A loop that repeats specific code a certain number of times before moving on.
  • Wait ms: Pauses the robot's brain for a specific number of milliseconds (1000 ms = 1 second).

Robot Movement

  • Move at speed: Starts MIGO's motors. Speed goes from 0 (stopped) to 255 (maximum speed).
  • Stop Moving: Immediately stops all motors. Remember to use this to prevent MIGO from running away forever!

Sensors

  • Ultrasonic Distance: Returns the distance (in cm) of objects in front of MIGO.
  • IR Sensor: Returns True if it sees a dark line, False if not.
  • Button: Returns True if pressed.
  • Knob / Light Level: Automatically converts raw electricity into a smooth 0-100% scale.
  • Read Digital/Analog: Generic blocks for custom sensors! Digital returns True/False. Analog returns raw 0-4095.

Hardware Plugs

  • Set Plug Mode: Always put this in 'Setup'! Tells the brain if a plug is an Input (sensor) or Output (light/buzzer).
  • Write Digital: Sends full power (High) or no power (Low) to an output plug.
  • Write Analog: Sends partial power to an output plug using PWM (from 0 to 255). Great for fading LEDs or controlling speed.

Functions & Print

  • Define Function: Groups blocks into a custom reusable command (e.g. 'Dance').
  • Call Function: Runs the custom command you created.
  • Print to monitor: Sends data back to your screen. You can plug a sensor block inside it to view live readings in the bottom-left window!
